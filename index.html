Um novo modo de criar suas senhas

Bem-vindo ao SenhaFácil

Agora é possível você criar uma senha criptografada a partir de uma palavra qualquer. Para criptografar clique no botão criptografar, para voltar à palavra que você escolheceu, basta clicar em descriptografar. É necessário escolher um número de deslocamento para sua senha - guarde esse número, pois ele é a chave/segredo caso você a esqueça. O uso de caracteres maiúsculas e minúsculas devem ser utilizados. Divirta-se!
